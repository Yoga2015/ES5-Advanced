每一个 类 中 都必须有一个 constructor( ) 方法。

当 类中 如果没有显式定义，一个空的 constructor（）方法 会被默认添加。

constructor( ) 方法 用于 初始化 新创建的对象。

constructor( ) 方法 可以接受 参数，这些参数 用于 初始化 对象的属性。

在 constructor( ) 内部，可以使用 this关键字 来引用 当前实例 。

如果 类 继承 自另一个类，可以使用 super关键字 来调用 父类的constructor( ) 方法。
