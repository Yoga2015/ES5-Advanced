在 ES5 及之前的版本中，JavaScript 中 使用 构造函数 来创建 对象， 然后 通过 new关键字 来调用，从而 创建 并 初始化 新对象。

什么是 class

ES6 引入了 Class (类) 这个概念，作为 对象 的 模板 。

通过 class 关键字，来 定义 类 ，然后 也是通过 new关键字 来调用，从而 创建 并 初始化 新对象。

class 底层 是基于 ES5中 已有的原型继承机制 实现 的 语法糖。

为什么出现会 class

ES6 中 引入 的 class 主要是为了 提供一种 更接近于 传统面向对象编程语言 的 语法结构， 使得 JavaScript 的 面向对象编程 更加 简洁、清晰、易读。

使用 class 语法 可以让 对象原型 的 写法 更加 简洁、清晰、易读，更符合 面向对象编程 的习惯。
