for 循环 是 js提出时 就有的 循环方法。 forEach 是 ES5 提出的，挂载在 可迭代对象原型上 的方法，例如 Array、 Set 、Map。 forEach 是 一个迭代器，负责遍历 可迭代对象 （ Array、 Set 、Map等）。 forEach 与 for 循环本质上的区别： forEach 是 负责遍历（Array、Set、 Map）可迭代对象 的 for 循环 是 一种循环机制，只是能通过它遍历出数组 。 forEach 循环 可以遍历 所有实现 Iterable 接口 的 所有对象 ，所以只要让我们的类实现 Iterable 接口就可以了 。 forEach 也不是全能的 ， forEach() 方法 没有 返回值。
